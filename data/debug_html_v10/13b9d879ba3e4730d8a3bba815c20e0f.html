PREFERÊNCIAS DE COOKIES

Cookies são conjuntos de dados que são armazenados em seu computador ou dispositivo móvel quando você utiliza nossos serviços. Eles são utilizados principalmente para fornecer ao usuário a melhor experiência possível de navegação, como por exemplo guardar informações que servem para identificar o visitante, seja para personalizar a página de acordo com o seu perfil ou para facilitar o transporte de dados entre as páginas, assim como providenciar alguns recursos essenciais para o bom funcionamento do site.

Mais Informações
GERENCIAR PREFERÊNCIAS DE CONSENTIMENTO
Cookies obrigatórios (Sempre Ativos)
"PHPSESSID" - Id da sessão

Utilizado para manutenção da sessão do usuário no sistema. Esse cookie ficará armazenado enquanto durar a sessão do usuário no sistema.

"cidade" - Identificação Ambiente Cliente

Utilizado para identificação pelo sistema da entidade que está sendo acessada pelo usuário. Esse cookie ficará armazenado enquanto durar a sessão do usuário no sistema.

"aceitouCookies" - Configuração das Preferências de Cookies

Responsável por manter a informação de preferências por parte do usuário quanto a utilização de cookies do sistema no navegador.

"votoEnquete" - Voto de Enquete

Utilizado para definir se o usuário já realizou alguma votação em determinada enquete no portal do cidadão layout dinâmico.

Cookies não obrigatórios
"WPO_USER" - Forma do Último Login do usuário Utilizado para salvar os dados de CPF/CNPJ do último login de usuário, para que em um próximo acesso ao sistema, no momento de login seja sugerido o CPF/CNPJ.
"AssinaturaDigitalCerts" - Cache de Certificados Digitais Disponíveis

Listagem contendo os certificados identificados como disponíveis para assinatura digital pelo usuário. Trata-se de um cache, evitando carregamentos desnecessários de certificados, aprimorando a performance durante assinaturas digitais de documentos. Esse cookie ficará armazenado enquanto durar a sessão do usuário no sistema.

"avisosPortal" - Avisos dos portais

Utilizado para definir os avisos que foram lidos pelo usuário, possibilitando a validação para que o mesmo não volte a ser exibido.

"_ga*" - GoogleAnalytics

São utilizados para coletar dados estatísticos de navegação para melhorar a experiência de uso, pois nos permite visualizar informações sobre a quantidade de visitantes, quais páginas são mais acessadas, sem identificar ou tornar o usuário identificável. Para saber mais sobre a utilização destas informações pelo Google acesse a Política de Uso de Dados do Google Analytics e também pela Política de Privacidade e Termos do Google.

Cookies não obrigatórios, mas importantes
"CookiePreferencia" - Preferências de cookies

Utilizado para salvar as informações de aceite e rejeição de cookies do sistema

"_grecaptcha" - ReCaptcha Google

Utilizado pelo serviço Google que valida se o usuário da sessão não é um robô, de modo a fornecer análise de risco.

"avisoBloqueadorWpo" - Aviso Bloqueador do Portal do Cidadão

Utilizado para definir se os avisos bloqueadores do Portal do Cidadão já foram previamente visualizados pelo usuário.

"registroNewsletter" - Registro de Newsletter

Utilizado no portal do cidadão para identificar se o usuário utilizador do sistema realizou previamente o cadastro de seu e-mail no elemento Newsletter;

"acessibilidadePortal" - Acessibilidade Portal

Utilizado para identificar as preferências de acessibilidade do usuário como tamanho da fonte e se utiliza o tema de autocontraste 

"avisoPendencias" - Aviso de Pendências do Usuário

Utilizado para definir se os avisos de pendências cadastrais do usuário no Portal de Autoatendimento já foram previamente visualizadas pelo usuário.
