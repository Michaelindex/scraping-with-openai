Foi em 2018, no mês de setembro, que a Ancre foi inaugurada, com o objetivo de fazer bem para a vida dos nossos pacientes.

A Ancre surgiu da idéia da Dra Julia e da Fernanda de ter uma clínica com uma equipe de profissionais altamente capacitados de diferentes áreas, preparados para atender todas as demandas relacionadas a saúde mental e emocional.

E tem coisa melhor do que ser reconhecida por fazer bem para vida das pessoas?

É isso que nos move!

PARA QUEM É A ANCRE?

A Clínica Ancre foi feita para mim. você. ele. ela. elu.

  • Para quem quer conseguir dizer não
  • Para quem quer dar um passo e precisa de orientação
  • Para quem quer regular as emoções
  • Para quem quer se alimentar melhor
  • Para quem sente dor
  • Para quem sente dor e não sabe explicar ao certo onde
  • Para quem se leva muito a sério e também para quem está precisando se levar um pouco mais
  • Para quem acredita que precisa de um suporte emocional
  • Para os casais, para quem está tentando se encontrar
  • E também para quem não sabe o que tem, mas sabe que algo não está legal
