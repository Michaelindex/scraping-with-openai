Guia Médico

O Guia Médico está disponível para consultas, visando facilitar o acesso da sociedade aos médicos. Para o Cremesp, o acesso do público a esses dados tem como objetivo identificar os profissionais e promover a comunicação entre médicos e pacientes, contribuindo para o fortalecimento desta relação com base na confiabilidade das informações.
